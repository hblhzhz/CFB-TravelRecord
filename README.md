# 出发吧-旅行打卡开销记录

出发吧-旅行打卡开销记录是一款为旅游出行打卡开销而设计的软件，旅游出行可以有计划，可以按旅行计划添加多个打卡地点，
完成计划的同时打卡记录美好点滴。同时可以按打卡地点记录旅行花销，还提供不同旅行计划的详细花销对比，旅行花销有个数，不超额，合理计划旅行开支。软件也支持分享/种草/拔草自己旅行所见所感，随时分享旅行中发生的点滴
